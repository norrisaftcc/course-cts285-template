# Agents

Exportable definitions for the custom subagents used on this project. Each file
captures a single agent's **original prompt** (as it currently lives in
`~/.claude/agents/`) alongside a **self-critique** and a **revised prompt** that
the agent generated by analyzing its own definition.

The goal is portability: the revised prompts are written to work in projects
beyond this course, with hardcoded course/project assumptions removed and
explicit discipline added for grounding, output, and scope.

## File structure

Each agent file contains:

1. **Header** — agent ID, source path, model, description.
2. **ORIGINAL PROMPT** — the verbatim system-prompt body currently deployed.
3. **SELF-CRITIQUE** — the agent's own assessment of its weaknesses.
4. **REVISED PROMPT** — the improved, more portable system-prompt body.

## How these were produced

Each agent was invoked with its own live definition and asked to critique and
rewrite its own system prompt for a future, exportable version. The revised
prompts are drafts for review — they have **not** been written back into
`~/.claude/agents/`. To adopt one, copy its **REVISED PROMPT** body under the
existing YAML frontmatter in the corresponding source file.
